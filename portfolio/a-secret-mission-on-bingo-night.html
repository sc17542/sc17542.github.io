A Secret Mission on Bingo Night

A Secret Mission on Bingo Night is an audio adventure idea for the mobile game "Zombies, Run!" The story is an action comedy about a secret agent grandma who must juggle between stopping an evil yarn bombing plot involving actual explosives and winning a bingo game where the grand prize could help stop that very plot.

This was created for an Experimental Game Narratives class at NYU in the Fall 2020 semester. We had to come up with a pitch, write out a partial script, create an audio recording of that script, and design screen mockups of this idea in "Zombies, Run!"
